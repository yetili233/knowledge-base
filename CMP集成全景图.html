📋 流程层 · ITSM
ServiceNow
全球最主流 ITSM 平台
工单审批CMDB变更管理
发起 VM/DB 申请 → 多级审批 → 触发 CMP 开通 → 完成后回写 CMDB CI 记录
对接：Table API (REST) · OAuth2 · Webhook 回调 · Import Set API
BMC Helix / Remedy
传统企业 ITSM
工单资产管理
老牌 ITSM，部分金融/国企仍在使用，功能与 ServiceNow 类似
对接：REST API · AR System SOAP API
🖥️ 资源层 · 基础设施
TFE · Terraform Enterprise
IaCVM创建网络/存储
声明式编排资源，CMP 触发后由 TFE 调用底层 vCenter/云 API 建资源
对接：POST /api/v2/runs 触发 → 传变量 → GET 轮询状态 → 读 Outputs
VMware vCenter
VM克隆资源调配
实际执行 VM 克隆、CPU/内存/磁盘配置，通常由 TFE vSphere Provider 调用
对接：vSphere REST API (7.0+) · pyvmomi SDK（老版本）
🌐 网络层 · DNS/IP
Infoblox
IPAM / DNS / DHCP 一体化
IP分配DNS注册网络资产
VM 开通时从 Pool 分配 IP，注册 A/PTR DNS 记录；VM 销毁时回收
对接：WAPI REST API · POST /wapi/v2.x/record:host · DELETE 回收
⚙️ 执行层 · 自动化
AAP · Ansible Automation Platform
配置管理DB安装Agent部署
核心执行引擎：OS 加固、DB 安装、中间件部署、监控/备份 Agent 安装
对接：POST .../job_templates/{id}/launch · 传 extra_vars · GET 轮询 Job 状态
Jenkins / GitLab CI
CI/CD应用部署
VM 开通后可触发 CI/CD 流水线完成应用自动部署（扩展场景）
对接：Jenkins REST API · GitLab Pipeline API / Webhook
🔐 安全层 · 凭证/PAM
HashiCorp Vault
秘密管理平台
密码动态下发证书Token
CMP/AAP 运行时动态拉取凭证，避免硬编码；支持自动轮换和完整审计
对接：AppRole 认证 → KV v2 REST · Ansible Vault lookup plugin
CyberArk
特权账号管理 PAM
root账号密码轮换跳板登录
VM 开通后注册 root/admin，运维人员通过 PSM 跳板登录，不直接接触密码
对接：PVWA REST API · POST /api/Accounts 创建 · DELETE 注销
📊 监控 & 备份层
Dynatrace / Datadog
APM基础设施告警
AAP 安装 OneAgent/Agent，自动发现主机进程，关联告警策略和 SLA 标签
对接：Dynatrace REST API 打标签 · Datadog POST /api/v1/tags/hosts
Splunk
日志采集SIEM
AAP 安装 Universal Forwarder，指向 Indexer，采集系统/应用/安全日志
对接：Splunk REST API · AAP Playbook 安装 Forwarder
Cohesity / NetBackup
备份策略AgentTag-based
按 SLA 标签选择备份策略，安装 Agent 注册保护组；VM 销毁自动解除
对接：Cohesity REST API · NetBackup REST (9.x+) · 或 AAP CLI
